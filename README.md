# alienworlds-claim-tool
alienworlds claim is low memory tool for alienworlds,it can monitor more than 100 account,and auto captcha and claim.

外星世界游戏辅助工具，基于golang开发的客户端，django开发的后台。通过后台自动调度账号达到同时挂几百个账号的目的。
